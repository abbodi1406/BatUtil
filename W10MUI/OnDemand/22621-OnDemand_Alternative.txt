- Starting build 19041, more OnDemand Packs are needed to complete language localization along with Lang Packs

- For each language, open the specified link in your web browser, then download the listed required files

- You also need to download rename_script.cmd file, or you can use the same file downloaded with LPs
- Place the downloaded files next to the script and run it
- Afterwards, move the files into the correct folder architecture (x64)

- LanguageFeatures-Basic is required for other LanguageFeatures packs to work
- LanguageFeatures-TextToSpeech is required for LanguageFeatures-Speech pack to work
- zh-HK packs require zh-TW LangPack

- If you found multiple cab files for the same pack, download the largest cab file
