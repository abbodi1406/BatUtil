# aria2 Downloader for UUPdump

A helper script to easily download multiple files from uupdump.net respecting rate limits

## How To

- Download aria2c.exe if not present
https://aria2.github.io/

- aria2c.exe will be detected from the script directory or the system directories (\Windows, \System32, \SysWOW64)

- Copy or gather uupdump.net/getfile links for the needed files

- Paste the links in aria2_links.txt, one link per each line, and save the file

- Run aria2_uupdump.cmd

- The files will be downloaded and saved in the current script directory
