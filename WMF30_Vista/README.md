# WMF30_Vista

- Windows Managemet Framework 3.0 (KB2506146) Online Installer for Windows Vista

## How To

- Install WMF 3.0 prerequisites and reboot:  
```
WMF 2.0 (KB968930)
BITS 4.0 (KB960568)
.NET Framework 4.0 or later
```

- Extract all contents of the 7z file to a short simple path

- Copy **Windows6.0-KB2506146** msu or cab file to the extracttion folder (next to the scripts)

- Run the 1st and 2nd scripts in the named order as administrator  
you must reboot after 2nd script  
after restart, you must run the 4th script
