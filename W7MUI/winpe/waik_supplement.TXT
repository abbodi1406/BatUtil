http://www.microsoft.com/downloads/en/details.aspx?FamilyID=0aee2b4b-494b-4adc-b174-33bc62f02c5d

you can choose any language iso, all the internal winpe lang packs are the same.
however, it's best not to select one of (english, german, french, japaneese, korean, chineese)
because they have Server 2008 R2 IA64 LP files, which make the iso size larger and unneeded

- copy the Supplement iso file to this directory 
in order to be detected, make sure the file retains its name or at least contain the word "supplement"
